Project Objective

The goal of this assignment is to design and build a basic 3-page website using WordPress. The process includes planning using Figma, selecting an appropriate theme, customizing content, and deploying the site.

Theme Justification

Chosen Theme: Astra

I selected the Astra theme because it's lightweight, fast-loading, and easy to customize. It works well with Elementor and offers responsive designs. For a personal portfolio scenario, Astra provides the perfect structure to showcase my work and contact information in a professional and clear format.
